Export to Excel
1
Download your data backup
Click the button below to download your data as a JSON file. This contains all your entries.
2
Download the Python export script
A Python script converts your JSON into a beautifully formatted Excel file with colored headers, department sections, and totals.
3
Run the script
Open Terminal / Command Prompt and run:
python3 pearl_laundry_excel_export.py your_backup.json
Requires Python 3 + openpyxl: pip install openpyxl
✅ The Excel output includes: one sheet per month with items × days matrix, colored department headers, subtotals, grand totals, and a Summary sheet.
